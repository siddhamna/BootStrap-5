# Startup Landing Page — Bootstrap 5 Implementation

A pixel-perfect, fully responsive landing page built with **Bootstrap 5**, raw CSS transitions, and Google Fonts. This project replicates a modern startup landing template featuring custom interactive UI overlays, dynamic hover animations, and a full-viewport media showcase.

## 🚀 Key Features

* **Full-Viewport Hero Header:** A background canvas utilizing `background-size: cover` that dynamically scales to fill $100\text{vh}$ across desktop, tablet, and mobile displays.
* **Centered Social Overlays:** Clean, modern profile cards for the creative team featuring interactive, perfectly centered social action tags (`Facebook`, `LinkedIn`, `Google`) that appear smoothly on hover.
* **Interactive Portfolio Grid:** A "Featured Works" showcase using pure CSS transforms. Hovering over a project image triggers an elegant slide-up overlay displaying the category, title, and a rounded call-to-action button.
* **Split-Screen Design:** An asymmetric layout highlighting the corporate identity through high-contrast image panels and clean typography.
* **Responsive Framework:** Leverages Bootstrap 5's flexbox grid architecture to maintain component alignment and clear spacing on all screen dimensions.

---

## 🛠️ Tech Stack & Dependencies

* **Markup & Layout:** HTML5, Bootstrap v5.3.3
* **Icons:** Bootstrap Icons v1.11.3
* **Typography:** Google Fonts (*Poppins* — Weights: 400, 500, 600, 700, 800)
* **Styling:** Custom CSS3 (Flexbox positioning, transform scales, linear-gradient masks)
